Lists

To make unordered lists we can use the label ul or, to make ordered lists could use as well ol

Unordered lists:

  • This is one item in my list
    • This is a hierarchy unordered list
    • This is a hierarchy unordered list
  • This is another item

Ordered lists:

  1. This is one item in my list
    1. This is a hierarchy ordered list
    2. This is a hierarchy ordered list
  2. This is another item

We can mix both type of lists:

  • This is one item in my unordered list
    1. This is a hierarchy ordered list
    2. This is a hierarchy ordered list
  • This is another item
  1. This is one item in my ordered list
    • This is a hierarchy unordered list
    • This is a hierarchy unordered list
  2. This is another item

Git commands

We can use git status in order to to know if there is something to commit to the repo.
We can use git add . in order to prepare all the changes to be commited.
We can use git commit -m"" to commit the changes.
We can use git push to push all the commits in to the repo.
